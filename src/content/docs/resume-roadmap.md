## Data Scientist Role

- Data Science involves using scientific methods, processes, algorithms and systems to extract knowledge and insights from structured and unstructured data
- It combines various disciplines such as statistics, machine learning, data analysis, and visualization to uncover hidden patterns, trends, and correlations in data
- Roadmap:
  1. Mathematical
  2. Probability
  3. Statistics
  4. Feature Engineering
  5. data visualization
  6. ML
  7. Deep Learning
  8. NLP
- https://www.geeksforgeeks.org/blogs/data-scientist-roadmap/
- https://roadmap.sh/ai-data-scientist

## Machine Learning Role

- Machine Learning (ML) is a subset of Artificial Intelligence (AI) focused on building algorithms that allow computers to learn from data and make predictions
- Roadmap:
  1. Machine Learning Beginner Level
     1. Machine Learning Algorithms for Supervised, Unsupervised, Reinforcement learning
  2. Machine Learning Intermediate Level
     1. Model Selection
     2. Model Evaluation and Tuning
  3. Machine Learning Advanced Level
     1. Deep Leaning
     2. NLP
     3. CV
- https://www.geeksforgeeks.org/blogs/machine-learning-roadmap/
- https://roadmap.sh/machine-learning

## Deep Learning Role

- Deep Learning refers to a type of machine learning that utilizes neural networks with many layers (thus, 'deep') to handle complex representations of data
- Roadmap:
  1. Math: Linear Algebra, Vector Calculus, Probability, Statistics
  2. Frameworks: Popular ones are TensorFlow, PyTorch, Keras, and FastAI
  3. ML: Key Algorithms, Model Evaluation
  4. Neural Networks: Activation Functions, Feedforward Networks, Back Propagation
  5. Artificial Neural Networks (ANN)
  6. Convolutional Neural Networks (CNN)
  7. Recurrent Neural Networks (RNN)
  8. Generative Adversarial Networks (GAN)
  9. Long Short-Term Memory (LSTM)
  10. Autoencoders
  11. Transformers
- https://www.geeksforgeeks.org/blogs/deep-learning-roadmap/

## Prompt Engineer Role

- Prompt engineering refers to the process of designing and crafting effective prompts for language models like ChatGPT
- Prompt engineering involves considering factors such as:
  - desired output format
  - specific information to include or exclude
  - desired style or tone
  - any additional constraints or requirements
- Roadmap:
  1. Understand the Basics of NLP
  2. Explore NLP Libraries and Frameworks: Natural Language Toolkit (NLTK), spaCy, and Transformers
  3. Python package: PyTorch
  4. Understand ChatGPT and Transformer Models
- https://www.geeksforgeeks.org/blogs/roadmap-of-becoming-a-prompt-engineer/
- https://roadmap.sh/prompt-engineering

## Generative AI Role

- Generative AI (GenAI) is transforming various fields by enabling machines to create text, images, videos, and more
- Roadmap:
  1. Solid understanding of foundational concepts in Data Science and Machine Learning (ML)
  2. Generative AI Key Concepts:
     - Transformers
     - GANs (Generative Adversarial Networks)
     - Autoencoders
     - Variational Autoencoders (VAE)
     - Attention Mechanism
  3. Introduction to LangChain in Generative AI:
     - Working with LLMs
     - Monitoring with LangSmith
  4. Vector Stores and Embeddings:
     - Vector Databases
     - Text Embeddings
     - Building a semantic search engine
  5. Prompt Engineering for GenAI
  6. Working with LlamaIndex for GenAI Applications (RAG)
  7. Fine-Tuning Large Language Models (LLMs)
- https://www.geeksforgeeks.org/artificial-intelligence/roadmap-to-generative-ai-a-comprehensive-guide-for-beginners/
- https://roadmap.sh/ai-engineer
